MMA Nepal — Event & News Platform
A full-stack web application built with Python and Flask for managing and displaying MMA events, fighter profiles, and news with a special focus on Nepali fighters and events.

Features
1. Public Pages

Homepage with event poster hero background and live countdown timer (Nepal timezone)
News page with category filters (UFC, Nepali, etc.)
Fighter profiles with full records (W/L/D), win methods, and social media links
Nepali fighter spotlight section with Nepal badge
Events page with upcoming and past events

2. Admin Panel

Role-based authentication (admin/user)
Full CRUD for fighters, events, news posts, and fight cards
File uploads for fighter photos and event posters
Automatic fighter record updates after fight results are marked
